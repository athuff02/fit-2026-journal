# Fit 2026 Mindset Journal

This is a simple, offline-first journaling application designed to help you focus on your goals and personal growth. It's a progressive web app (PWA), so you can install it on your mobile device and use it like a native app.

## User Instructions

1.  **Open the application:** Simply open the `index.html` file in your web browser.
2.  **Daily Focus:** The application will present you with a daily theme and a set of questions to guide your journaling.
3.  **Save Your Entry:** Fill out the questions and the action item, then click "Save Entry." Your entry will be saved to your device's local storage.
4.  **History:** You can view your past entries in the "History" tab. You can also filter your entries by theme.
5.  **Export:** You can export individual entries as a text file from the history view.

## Technical Overview

This application is built with HTML, CSS, and JavaScript. It uses the following technologies:

*   **Tailwind CSS:** For styling the user interface.
*   **IndexedDB:** For secure, client-side storage of your journal entries.
*   **Progressive Web App (PWA):** The application can be installed on your mobile device for an offline-first experience.

All of the application's logic is contained in the `app.js` file. The code has been commented to make it easy for novice developers to understand.
